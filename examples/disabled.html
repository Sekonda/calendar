Calendar Disabled Days/Dates
  • « Action
  • ^ Example Index

You can disabled specific days of the week, or specific dates, or a mixture of both.

disabled_days array An numeric (0-6) array of days of the week that should be disabled
disabled_dates array An array of dates that should be disabled.
<div id="my-calendar"></div> <script type="application/javascript"> $(document).ready(function () { $("#my-calendar").zabuto_calendar({ disabled_days: [0, 6], disabled_dates: ['2018-12-31', '2019-01-01'] }); }); </script>
